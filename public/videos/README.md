# Vidéos du Chef — À intégrer

Déposer ici les fichiers vidéo du Chef Régis Clauss en cuisine.

## Spécifications techniques

| Fichier               | Format | Codec  |
|-----------------------|--------|--------|
| chef-cuisine.mp4      | MP4    | H.264  |
| chef-cuisine.webm     | WebM   | VP9    |

## Recommandations
- **Orientation** : Portrait (ratio 3:4 ou 4:5) — ex : 1080×1350px
- **Durée** : 5 à 25 secondes, boucle parfaite
- **Audio** : Muet (autoPlay muted requis pour l'autoplay navigateur)
- **Poids** : < 8 Mo idéalement (compressé avec HandBrake ou FFmpeg)
- **Contenu** : Chef en cuisine, gestes culinaires, dressage de plat

Si les fichiers sont absents, la photo `chef-portrait.avif` s'affiche automatiquement.
